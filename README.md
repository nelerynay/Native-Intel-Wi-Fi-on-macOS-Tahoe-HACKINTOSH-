### Native Intel Wi-Fi on macOS Tahoe

This is a project about how to get native Intel Wi-Fi on macOS Tahoe, a workaround project and not official guide from [OpenIntelWireless](https://openintelwireless.github.io/)

### Compatibility

- Supported Intel Wi-Fi module from [OpenIntelWireless compatibility page](https://openintelwireless.github.io/itlwm/Compat.html)
- Hackintosh running macOS Tahoe
